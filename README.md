# 🧟 Ty vs Zombie 2

> A fast-paced first-person zombie shooter built with **Godot 4.7**.

Fight through hordes of undead using a growing arsenal of weapons, explore interactive environments, drive vehicles, and survive increasingly dangerous encounters.

---

## 🎮 Features

- 🧟 Zombie AI with patrol, chase and attack behavior
- 🔫 Multiple weapons with unique gameplay
  - Knife
  - Pistol
  - Assault Rifle
  - Burst Rifle
  - Light Machine Gun
- 🎯 Hitscan weapon system
- 💥 Exploding barrels
- 🚙 Drivable vehicles
- 🚪 Interactive doors
- ❤️ Health and damage system
- 📦 Ammo pickups
- 🔊 Dynamic sound effects
- ✨ Visual effects (muzzle flashes, explosions, bullet decals)
- 🖥️ Complete FPS HUD
- 🎮 Smooth first-person movement
  - Sprint
  - Jump
  - Crouch
  - Aim Down Sights (ADS)

| Action | Key |
|---------|-----|
| Move | WASD |

| Aim | Right Mouse Button |

## 🚀 Getting Started

### Requirements

- Godot Engine **4.7**

### Running the Project

1. Clone the repository

```bash
git clone https://github.com/KtosNapewno95/Ty-vs-Zombie.git
```

2. Open the project using Godot 4.7

3. Press **F5** or **Fn+F5** to play.
